# Plan: Mouse Jiggler Toggle (based_vim)

## Summary
- Add a custom keycode toggle on the Fn (UTIL) layer top-right blank key.
- Default off on boot; when enabled, send a tap of F15 every 30 seconds.
- Update keymap comments to document the new toggle per AGENTS.md.

## Files to change
- `keyboards/zsa/voyager/keymaps/based_vim/keymap.c`

## Implementation steps
1. Add a new custom keycode (<=7 chars, e.g., `JIGGLR`) to the custom keycodes enum.
2. Add static state in `keymap.c`:
   - `static bool jig_on = false;`
   - `static uint32_t jig_tim = 0;`
3. Place `JIGGLR` on the UTIL layer at the top-right blank key position and update the ASCII layout comment to describe it as the jiggler toggle.
4. In `process_record_user`, handle `JIGGLR`:
   - On key press, toggle `jig_on`.
   - When enabling, set `jig_tim = timer_read32()`.
   - Return `false` for the keycode.
5. Implement `housekeeping_task_user` in `keymap.c`:
   - If `jig_on` and `timer_elapsed32(jig_tim) >= 30000`, call `tap_code(KC_F15)` and reset `jig_tim`.

## Verification
- Build the keymap: `qmk compile -kb zsa/voyager -km based_vim`.
- Flash and verify:
  - Toggle key on UTIL layer enables/disables jiggling.
  - When enabled, F15 is tapped every ~30 seconds.
  - Toggle off stops the taps.
